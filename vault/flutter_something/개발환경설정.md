## MacOS

| Homebrew를 통해 할 수 있었으면 좋겠지만 구형 맥이라 MacOS 12 상태로는 불가능
| 집에 있는 Macmini m4 깡통은 그냥 homebrew로 가볍게, extend로 가볍게 설정 가능하다

### 설치하기
```zsh
brew update
brew upgrade
brew install flutter
which flutter
# [/opt/homebrew/bin/flutter] 나오는게 정상 아니면 [export PATH="$PATH:/opt/homebrew/bin"]
brew install android-studio
# android-studio 열기 -> Setting(command + ,) -> Languages & Frameworks -> Android SDK -> SDK Tools -> Android SDK Command-line Tools (latest) -> Apply -> OK
Flutter doctor
# 경고 있을것임 -> Some Android licenses not accepted. To resolve this, run: flutter doctor --android-licenses
flutter doctor --android-licenses # 당연히 다 동의해주면 된다.
```
